About

The idea for BoardgameBro came about around 2015.
It wasn't until 2019 the site actually launched when Gabbeh recently recovered the idea.
BoardgameBro is aiming for alpha status Q1 2019, with MVP status for all basic functionality.
The hope is to create some cool status both globally and for your gamerfriends.
To create cool stats, it has to be easy to add data, so this project aims for a high end ux experience.
